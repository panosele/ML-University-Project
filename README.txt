Requirements:

	1. Python (>= 3.10) (Installation : https://www.python.org/downloads/)

	2. pip    (>=22.2.1)  (Installation : https://pip.pypa.io/en/stable/installation/) (Needed to install python packages)

  		To install the packages use "pip install 'package'" ( How to do it : https://pip.pypa.io/en/stable/user_guide/)

	3. matplotlib (>=3.5.1)

	4. NumPy (>= 1.22.2)

	5. scikit-learn (>=1.0.2) (Installation : pip install -U scikit-learn) requires:
		SciPy (>= 1.8.0)
		joblib (>= 1.1.0)
		threadpoolctl (>= 3.1.0)

	6. tkinter (standar library)

	7. pandas (standar library)
	


You can install these packages, in cmd you can navigate to the folder of the project and run the app : python3 main.py 
	or
You can download PyCharm and run it from the enviroment. (https://www.jetbrains.com/help/pycharm/installation-guide.html)
